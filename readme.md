A Monogame implementation of the Separating Axis Theorem for convex polygon intersection.

---

To generate the necessary `.xnb` files, run:

```bash
$ mgcb --outputDir=bin/Debug/netcoreapp3.1 --build=Content/textures/circle.png
```
